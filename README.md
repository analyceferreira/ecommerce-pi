# Running the app
```bash
# clone repository
$ git clone https://github.com/analyceferreira/ecommerce-pi.git

# install dependes
$ npm install

# run app
$ npm star
```
<br>

# Database config
Use with databe config rename .env.exemple file to .env and insert your database informations
